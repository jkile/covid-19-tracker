# Covid-19 Tracker
 
COVID-19 tracker:
* db of cases/tests by state and date
* separate page/view to add data to db
* view data by day and/or state
* cases added will go to secondary view of "unverified" cases for self reporting illness


Technologies:
* Anime.js
* d3
* Sequelize
* Node
* Express
* MySQL
* Handlebars
* Tailwind

Project roles: 


Data source:
